# 學習清單 APP (Learning List Android App)

本專案為行動裝置程式設計課程之期末作品，使用 Java 語言與 Android Studio 開發。開發動機是為了解決一般學習應用程式功能過於繁瑣、且受限於日期規劃壓力等問題，提供一個以「百分比進度條」為核心的輕量化管理工具，幫助使用者在低壓力的環境下養成學習習慣。

## 專案演示
展示影片連結：[https://youtu.be/mlsk8V5BQQ4](https://youtu.be/mlsk8V5BQQ4)

## 核心功能
* **目標管理**：支援學習目標的新增、編輯與刪除。
* **進度可視化**：以百分比取代傳統日期規劃，提供 +10%/-10% 按鈕，搭配視覺化進度條即時呈現成就感。
* **檔案管理**：具備課程相關檔案之上傳與刪除功能，便於整合學習資源。

## 技術實作與個人貢獻
在團隊開發中，我主要負責「編輯學習目標」模組的開發與整體流程規劃，具體實作細節如下：

1. **資料交互設計**：
   - 運用 **Intent** 實作 Activity 之間的數據傳遞，確保主介面的目標資訊能準確載入至編輯頁面進行修改。
2. **邏輯控制與防錯**：
   - 撰寫進度增減的判斷邏輯，確保進度數值嚴格維持在 0% 至 100% 之間，並同步更新資料庫數據。
   - 實作 **AlertDialog**（刪除確認對話框）防呆機制，避免使用者誤觸導致資料流失。
3. **介面佈局優化**：
   - 負責編輯頁面的XML佈局規劃，確保輸入框、進度調整按鈕在不同螢幕尺寸下皆能保持直觀的操作體驗。

## 開發技術棧
* **開發語言**：Java
* **開發環境**：Android Studio
* **資料持久化**：SQLite Database (用於儲存學習目標與進度資料)
* **核心元件**：Intent、Custom Adapter、AlertDialog、ProgressBar
