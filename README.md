# 學習清單 APP (Learning List Android App)

本專案為行動裝置程式設計課程之期末作品，使用 Java 語言與 Android Studio 開發。開發動機是為了解決一般學習應用程式功能過於繁瑣、且受限於日期規劃壓力等問題，提供一個以「百分比進度條」為核心的輕量化管理工具。

## 專案演示
展示影片連結：[https://youtu.be/mlsk8V5BQQ4](https://youtu.be/mlsk8V5BQQ4)

## 核心功能
* **目標管理**：支援學習目標的新增、編輯與刪除。
* **進度追蹤**：提供 +10% / -10% 按鈕，搭配視覺化進度條即時呈現完成度。
* **檔案整合**：具備課程檔案上傳與清單顯示功能，便於學習資源管理。

## 技術實作與個人貢獻
在團隊開發中，我主要負責「編輯學習目標」模組的開發，具體實作細節如下：

1. **資料交互設計**：
   - 運用 Intent 實作 Activity 之間的數據傳遞，確保主介面的目標資訊能準確載入至編輯頁面。
2. **邏輯控制與防錯**：
   - 撰寫進度增減的判斷邏輯，確保進度數值嚴格維持在 0% 至 100% 之間。
   - 實作刪除確認對話框（AlertDialog），避免使用者因誤觸而導致資料流失，提升操作安全性。
3. **介面排版**：
   - 負責編輯頁面的佈局規劃，確保輸入框與按鈕在不同螢幕尺寸下的呈現符合操作邏輯。

## 開發技術棧
* 開發語言：Java
* 開發工具：Android Studio
* 資料管理：SQLite 資料庫、Shared Preferences (依實作細節調整)
